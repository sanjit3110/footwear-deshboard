# footwear-deshboard
Project Title
Footwear Sales Dashboard using Power BI

Description
I created this Power BI dashboard to analyze footwear sales data and understand overall business performance. The dashboard helps in tracking sales, quantity, transactions, and average price.
It also shows trends over time and compares current performance with the same period last year, which helps in making better business decisions.

Tech Stack
Power BI for dashboard creation
Power Query for data cleaning and transformation
DAX for calculations and measures

Data Source
The dataset is based on footwear sales data in CSV format. It includes fields like order date, brand, category, country, quantity, sales, payment method, and ratings from kaggle.

Features / Highlights
The dashboard includes KPI cards for total sales, quantity, transactions, and average price.
It has time-based analysis using year, quarter, month, and day.
There is a comparison of sales with the same period last year.
It shows sales by country using a map.
Brand-wise performance is included to compare different brands.
Payment method analysis is also shown.
Customer rating analysis is included to understand feedback.
Interactive filters are available for brand, category, and date.

Key Visuals
KPI cards for overall performance
Line chart for sales trend (MTD)
Bar chart for sales comparison with last year
Map for sales by country
Bar chart for sales by brand
Pie chart for payment methods
Area chart for sales by day
Chart for rating distribution
